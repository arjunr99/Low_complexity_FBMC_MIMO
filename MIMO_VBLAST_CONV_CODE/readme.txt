>> This program implements a MIMO with a V-BLAST architecture and half rate error correcting code..

>> the top files are main_EPA.m, main_EVA.m and main_ETU.m..
>> Run one of these to run simulation for the respective channels..

>> changing the spectral efficiency is done by changing one parameter in 3 files..
>> fbmc_fb_k32_33.m, fbmc_trans.m, fbmc_rec.m..
>> check fbmc_fb_k32_33.m,fbmc_trans.m and fbmc_rec.m to know what changes are needed.
